================================================================================
  G1 TALK MODULE - Installazione
  Assistente vocale per robot Unitree G1
================================================================================

IN 30 SECONDI — COME FUNZIONA
------------------------------
• Sul G1 (Linux) installi lo ZIP e avvii il SERVER: install.sh → .env → restart_server.sh
• Dal telefono: stesso WiFi del robot → browser su https://IP-DEL-G1:8081/client
  (oppure app APK: stessa pagina; inserisci l'IP; cassa BT = accoppiamento Android)
• Voce, GPT, TTS e soundboard girano SUL SERVER; il telefono è solo interfaccia.
• Pacchetto "tutto in uno" (Windows, cartella dist): G1_Pacchetto_Installazione_Completa.zip


INSTALLAZIONE (5 passi)
----------------------

1. Estrai il pacchetto
   unzip G1-TalkModule-OpenAiAPI.zip
   cd G1-TalkModule-OpenAiAPI

2. Installa
   bash install.sh

   Jetson G1 (computer di bordo) — installazione COMPLETA:
   bash scripts/install_jetson_completo.sh
   (include Unitree SDK movimenti + OpenCV visione; install.sh da solo non basta)

3. Configurazione
   Il pacchetto include .env con le chiavi API gia configurate.
   Se serve modificare: nano .env

4. Avvia il server
   bash scripts/restart_server.sh

5. Apri nel browser
   http://<IP-DI-QUESTA-MACCHINA>:8081/client

   Esempio: se l'IP è 192.168.1.100 → http://192.168.1.100:8081/client


COME USARE
----------

- Tieni premuto il pulsante e parla, rilascia per inviare
- Oppure scrivi nella casella di testo
- Domande tipo "che ore sono", "che tempo fa", "dammi la mano" funzionano subito


CONNESSIONE
-----------

Opzione A - Telefono, stesso WiFi (o WiFi del robot)
  Collega il telefono al WiFi del robot o alla stessa rete del server.
  Apri https://<IP>:8081/client (per microfono serve HTTPS)
  Oppure http://<IP>:8080/client (redirect a HTTPS)
  Launcher: http://<IP>:8081/launcher (inserisci IP e connetti)

Opzione B - Add to Home Screen (PWA)
  Da Chrome Android: menu → Aggiungi a schermata Home.
  L'app si comporta come un'app nativa.

Opzione C - APK Android
  Vedi capacitor-app/README.md per build dell'APK.

Opzione D - PC, tunnel SSH
  ssh -L 8081:localhost:8081 utente@IP-SERVER -N
  Poi apri http://localhost:8081/client


REQUISITI
---------

- Linux (Ubuntu, Debian, Jetson, Raspberry Pi)
- Python 3.9+
- Chiave API OpenAI (platform.openai.com)

Per mic/speaker locali sul server: PortAudio (install.sh lo installa).
Solo modalità web: bash install.sh --no-audio

PROBLEMI?
---------

- Server non parte: verifica OPENAI_API_KEY in .env
- Mic non funziona da browser: usa localhost (tunnel SSH) o HTTPS
